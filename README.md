# Blog Tecnológico - Proyecto Django

---

## Descripción

Aplicación web desarrollada con Django siguiendo el patrón **MVT (Modelo - Vista - Template)**.

Permite:

* Crear publicaciones
* Visualizar publicaciones
* Buscar publicaciones por título

---

## Estructura del proyecto

* Modelo: Publicacion
* Vistas: manejo de lógica (crear, listar, buscar)
* Templates: HTML con herencia
* Base de datos: SQLite

---

## Modelo utilizado

**Publicacion**

* titulo
* contenido
* autor
* fecha (automática)
